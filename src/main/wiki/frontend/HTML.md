# HTML

## `rel` and SEO
Тег `rel` використовується тільки в атрибуті `<a>`.

Може впливати на трафік Google Analytics.

Є декілька значень, наприклад `noreferrer`:
```js
<a href="https://cats.com" target="_blank" rel="noreferrer">Cats</a>
```
